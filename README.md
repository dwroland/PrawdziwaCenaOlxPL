Userscript do wtyczki [tampermonkey](https://www.tampermonkey.net/) naprawiający stronę [OLX.pl](https://www.olx.pl/) tak
aby pokazywała ona prawdziwą cenę wynajmowanych mieszkań.

![Podgląd na olx](preview.png)
